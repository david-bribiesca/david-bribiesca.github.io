Tema que conozco:

El tema que conozco es html y css, estos dos son lo más importante para el desarrollo web. El html es el lenguaje de marcado que se utiliza para estructurar el contenido de una página web, mientras que el css es el lenguaje de estilos que se utiliza para darle formato y diseño a la página web. Ambos son fundamentales para crear páginas web atractivas y funcionales.
